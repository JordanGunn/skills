# Reference Files

Reference files are kept in a `references/` subdirectory and follow the naming pattern:

- Format: `<NN>_<TOPIC>.md`
- `<NN>`: Zero-padded sequential number (00, 01, 02, ...)
- `<TOPIC>`: Descriptive name in uppercase with underscores

**Example structure:**

```text
skill-name/
├── SKILL.md
├── references/
│   ├── 00_INSTRUCTIONS.md
│   ├── 01_INTENT.md
│   ├── 02_PRECONDITIONS.md
│   └── 03_PROCEDURE.md
└── scripts/
    ├── script.sh
    └── script.ps1
```
